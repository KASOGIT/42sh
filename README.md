# 42sh

Shell on linux system that contains all features expected of such program (pipe/redirection/smart prompt/autocomplete/alias/key shortcut/..)

## Installation && Tools

Just need to run 'make' on linux system.</br>

## Usage

./42sh -> And enjoy !
